# NSLP Action Plan Card — Template

**Source:** SRC-MAYURIKA-NSLP-001 — HR.Mayu.Skill.md Section 9

**Purpose:** Collected from every participant before they leave the NSLP session. The Action Plan Card is mandatory — no participant leaves without submitting one. It records what the participant learned and how they commit to applying it.

**Safety note:** No real staff names, employee IDs, salary, health, grievance, disciplinary, or PDPA personal data. Use role/placeholder references only.

**Root Truth Rule:** If this template conflicts with HR.Mayu.Skill.md Section 9, the skill file is correct.
