## Column Definitions

| Column | Definition |
|---|---|
| Skill ID | Unique ID assigned to this new skill before the session begins. Must exist before the session starts. |
| Skill Name | Short name of the skill taught |
| BGCT Document Link | Path or reference to the BGCT document for this skill. Must exist before the session starts. |
| Trainer / Host | Role title of the person who ran the session (not personal name unless operationally required) |
| Department / Team | Team or department that attended |
| Meeting Date | Date the NSLP session took place |
| Attendees Recorded? | Y if attendee list is saved and cross-referenced to Action Plan Cards |
| Action Plan Cards Received? | Y if all participants submitted an Action Plan Card before leaving |
| Before Evidence Due | Date by which Before evidence must be collected (should be before or on Meeting Date) |
| After Evidence Due | Date by which After evidence must be collected (typically 2 weeks after meeting) |
| 2-Week Evaluation Date | Scheduled date for the formal 2-week evaluation |
| Adoption Status | Whether the participant has adopted the skill into daily work |
| Implementation Status | Whether the skill has been implemented with measurable output |
| Business Impact Summary | Brief description of the business impact, or "Pending" |
| HR Verification Result | PASS (evidence confirmed), FAIL (evidence not confirmed), or PENDING |
| Management Report Reference | Reference to the monthly report that includes this skill |
| Status | OPEN (active tracking), CLOSED (outcome confirmed), EXCEPTION (escalation raised) |
| Next Step | The immediate next action HR must take |

---

## Adoption Status Values

| Value | Meaning |
|---|---|
| Adopted | Skill is being used in daily work |
| Pending | Awaiting 2-week evaluation |
| Not Adopted | Skill not applied — exception to be raised |

## Implementation Status Values

| Value | Meaning |
|---|---|
| Implemented | Skill applied with measurable output confirmed |
| Pending | Evidence not yet collected |
| Not Implemented | No measurable output — FAILURE or NOT EFFECTIVE label applies |

## HR Verification Result Values

| Value | Meaning |
|---|---|
| PASS | HR has reviewed and confirmed Before & After evidence meets standard |
| FAIL | Evidence missing, opinion-only, or improvement not sustained |
| PENDING | Evidence not yet submitted or under review |
