# **Important Rules**

* Use only factual data.  
* Do not estimate values.  
* Clearly identify assumptions where data is unavailable.  
* Highlight all KPIs using color coding:  
  * 🟢 Target Achieved  
  * 🟡 Needs Improvement  
  * 🔴 Critical  
* Sort tables by highest business impact.  
* Ensure all tables are directly copy-paste compatible with Microsoft Excel and Google Sheets.  
* Include totals, percentages, and summary rows where applicable.  
* The report should be suitable for management review without requiring additional formatting.
