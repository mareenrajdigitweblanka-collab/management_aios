# HR Schedule Pilot — Archive

This folder holds superseded versions of HR schedule pilot files (weekly schedules, priority queues, recurring-block definitions) once they are replaced.

---

## Rules

- Move a schedule file here when it is replaced by a newer version — do not delete history.
- Name archived files with the week/date they covered, e.g. `mayurika-week-of-YYYY-MM-DD.md`.
- Archived files are historical records. Do not cite them as the current schedule.
- No real staff data, candidate personal data, salary, health, PDPA, or disciplinary detail belongs here (CLAUDE.md §6) — the pilot files are placeholder-only.
