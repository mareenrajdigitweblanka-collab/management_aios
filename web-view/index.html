Team Members at a Glance

How to use this dashboard

This dashboard gives you a quick view of the Management AIOS — a system that tracks management gaps, HR records, recruitment, and KPI processes. It is a read-only view. You cannot change anything from here.

  • Use the tabs above to move between different areas (Root overview, each team member's section, review actions, and the file map).
  • Use the search bar to find anything — file paths, names, or topics — across all tabs.
  • Editing is done in VS Code only, by opening the repository files directly.
  • Every tab has a "What should I do next?" box at the bottom — check that first if you're unsure what to do.
