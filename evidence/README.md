## Purpose

This folder holds validation proof, screenshots, logs, and review evidence for all approvals and decisions made in the LEDSone Management AIOS. Every time a context file is approved, a skill is authorised, or an intelligence item is processed, a record of that review event belongs here.

This is the audit trail for the system.

---

## What Belongs Here

- Review records: date, reviewer, item reviewed, outcome.
- Screenshots or logs from validation sessions.
- Signed-off checklists from `../validation/`.
- Skeleton build validation outputs (folder listings, git status).

## What Must NOT Be Stored Here Yet

- Sensitive personal HR data.
- Invented or assumed evidence.
- Approvals not traceable to a real review event.
- Any content from systems not yet connected or authorised.
